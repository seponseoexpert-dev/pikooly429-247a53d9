

# 🌸 Pikooly — Online Flower, Gift & Cake Shop

A gorgeous, fully responsive e-commerce website for flowers, cakes, and gifts in Bangladesh with stunning animations, SEO optimization, and fast performance.

---

## 1. Header & Navigation
- Sticky header with **Pikooly** logo, search bar, and cart icon with badge count
- Hamburger menu on mobile, full nav on desktop
- Smooth slide-in mobile menu with categories
- Bottom navigation bar on mobile (Home, Shop, Menu, Cart, My Account)

## 2. Hero Section
- Auto-sliding banner carousel with fade/slide animations (Anniversary Surprises, seasonal offers, etc.)
- Bold headlines with CTA "Order Now" buttons
- Dot indicators for slide navigation

## 3. Category Grid
- Scrollable category icons in a neat grid: Same Day, Premium, Perfumes, Makeup, Flowers, Combos, Cake, Birthday
- Rounded card design with subtle hover animations
- Star divider decoration between sections

## 4. New Arrivals & Product Listings
- Tabbed product sections (Small Flowers, Flowers, Birthday, etc.)
- Product cards with image, name, price in ৳ BDT, wishlist heart button, and "Add to Cart" button
- Smooth fade-in animations on scroll
- Horizontal carousel with arrows for featured collections (e.g., Tulip Bouquets)

## 5. Product Detail Page
- Large product images with gallery/zoom
- Product name, price, description, quantity selector
- "Add to Cart" and "Buy Now" buttons
- Related products section

## 6. Shopping Cart & Checkout
- Slide-out cart drawer showing items, quantities, and totals
- Full checkout page with delivery address form
- Order summary with delivery fee calculation
- Order confirmation page

## 7. User Authentication & Accounts
- Sign up / Login pages with email authentication
- "My Account" page with order history, saved addresses, and wishlist
- User profile management

## 8. Shop / Category Pages
- Full product listing with filters (price range, category, occasion)
- Sort options (price, newest, popularity)
- Pagination or infinite scroll

## 9. Blog Section
- "Fresh Insights & Flower Stories" blog area
- Blog post cards with date badge, category tag, author, and featured image
- Blog detail page with rich content

## 10. About & SEO Content Section
- "Pikooly: Online Flower Shop in Bangladesh" description block
- Rich, keyword-optimized content for SEO ranking
- Structured data and meta tags on every page

## 11. FAQ Section
- Expandable accordion-style FAQ (22+ questions)
- Smooth expand/collapse animations
- Questions about ordering, delivery, freshness, customization

## 12. Footer
- Tagline: "Not just a Gift. It's sharing of Love."
- Social media icons (Facebook, Instagram, X, TikTok, WhatsApp)
- Collapsible "Useful Links" and "Main Categories" sections
- Copyright notice and payment method icons
- WhatsApp floating chat button

## 13. Animations & Visual Polish
- Scroll-triggered fade-in and slide-up animations throughout
- Smooth page transitions between routes
- Hover effects on cards, buttons, and links
- Loading skeleton screens for product grids
- Subtle parallax on hero banners

## 14. SEO & Performance
- Semantic HTML with proper heading hierarchy (h1, h2, h3)
- Meta titles, descriptions, and Open Graph tags per page
- Lazy loading for images
- Optimized component code splitting
- Fast initial load with minimal bundle size

## 15. Backend (Supabase / Lovable Cloud)
- **Products table**: name, price, images, category, description, stock
- **Categories table**: name, icon, slug
- **Orders table**: items, delivery address, status, total
- **User profiles**: name, phone, addresses
- **Wishlist**: user saved products
- **Blog posts table**: title, content, author, date, category
- Row-level security for user data protection
- Storage bucket for product and blog images

## 16. Responsive Design
- Mobile-first design matching the Pikooly reference
- Tablet layout with 3-column product grid
- Desktop layout with 4-column grid and sidebar navigation
- Bottom nav on mobile, top nav on desktop
